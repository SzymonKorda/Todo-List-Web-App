API documentation:
https://todolist-backend.cytr.us/swagger-ui/

Application demo:
https://todolist.cytr.us/

Used technologies: 
- Java 11,
- Spring Boot, 
- Hibernate, 
- REST API,  
- jUnit 5, Mockito,
- Maven,
- JWT, 
- Project Lombok,
- Swagger
- Docker,
- PostgreSQL
