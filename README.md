# Todolist-backend
Backend for simple tasks managing web application.

Application demo:
https://todolist.skorda.pl/

API documentation:
https://api.todolist.skorda.pl/swagger-ui/#

Used technologies: 
- Java 11,
- Spring Boot, 
- Hibernate, 
- REST API,  
- jUnit 5, Mockito,
- Maven,
- JWT, 
- Project Lombok,
- Swagger
- Docker,
- PostgreSQL
